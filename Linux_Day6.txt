Commands:

	1."systemctl status apache2" - Shows the status of web server.
	2."sudo systemctl stop apache2" - Stops the server.
	3."sudo systemctl start apache2" - restarts the web server.
	4."sudo systemctl enable apache2" - sets in a automatic mode.
	5."sudo apt install apache2" - Installs the web server.
	6."sudo apt install openssh-server" - Installs the server
	7."sudo apt install telnetd" - installs telnet.
	8."systemctl status inetd" - checks the status.
	9."chmod a+r first.sh" - it gives read per to all.
	10."chmod u+wx first.sh" - gives write and exe per to user.
	11."sort filename" - sorts alphabetically 
	12."sort -r" - Reversorder
	13."ping 8.8.8.8" - pings the given ip address.
	14."mkdir script && cd script && pwd"- make dir as script changes dir to script and shows path.
	15."./first.sh"- execute the file.
	16."chmod g-r first.sh" - to remove the read permission of group.
